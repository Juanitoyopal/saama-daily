# Saama Daily

Plataforma web premium para bienestar funcional, productos Gano Excel originales, compra segura y acompañamiento personalizado en Colombia y Estados Unidos.

## Stack

- Next.js
- TypeScript
- React
- Tailwind CSS

## Desarrollo local

```bash
npm install
npm run dev
```

Luego abre `http://localhost:3000`.
